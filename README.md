# Hobby vs. Business: Tax Decision Tree

## Overview

This project demonstrates how a tax concept can be translated into a simple decision-making tool.

Many taxpayers earn income from activities they enjoy — such as crafts, tutoring, consulting, online sales, or other side activities. A common question is:

**"Is this a hobby or is this a business?"**

The answer depends on facts and circumstances, including the taxpayer's intent, activity level, recordkeeping, and profit motive.

This decision tree is designed as an educational tool to help organize the thought process.
